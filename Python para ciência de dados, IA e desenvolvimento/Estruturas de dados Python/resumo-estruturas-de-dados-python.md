# Resumo: Estruturas de Dados em Python

## Tuplas

* Em Python, geralmente usamos **tuplas** para agrupar dados relacionados.
* Tuplas referem-se a coleções ordenadas e imutáveis de elementos.
* As tuplas geralmente são escritas como elementos separados por vírgulas entre parênteses `()`.
* O senhor pode incluir strings, inteiros e floats em tuplas e acessá-los usando índices positivos e negativos.
* O senhor pode executar operações como combinação, concatenação e fatiamento em tuplas.
* As tuplas são imutáveis, portanto, o senhor precisa criar uma nova tupla para manipulá-la.
* As tuplas, chamadas de aninhamento, podem incluir outras tuplas de tipos de dados complexos.
* O senhor pode acessar elementos em uma tupla aninhada por meio de indexação.

## Listas

* As **listas** em Python contêm coleções ordenadas de itens que podem conter elementos de diferentes tipos e são mutáveis, permitindo o armazenamento e a manipulação versáteis de dados.
* Uma lista é uma sequência ordenada, representada por colchetes `[]`.
* As listas têm mutabilidade, o que as torna semelhantes às tuplas.
* Uma lista pode conter strings, inteiros e floats; o senhor pode aninhar listas dentro dela.
* O senhor pode acessar cada elemento de uma lista usando indexação positiva e negativa.
* Concatenar ou anexar uma lista resultará na modificação da mesma lista.
* O senhor pode executar operações como adicionar, excluir, dividir e assim por diante em uma lista.
* O senhor pode separar elementos em uma lista usando delimitadores.
* O **aliasing** ocorre quando vários nomes se referem ao mesmo objeto.
* O senhor também pode clonar uma lista para criar outra lista.

## Dicionários

* Os **dicionários** em Python são pares de chave-valor que fornecem uma maneira flexível de armazenar e recuperar dados com base em chaves exclusivas.
* Os dicionários consistem em chaves e valores, ambos compostos de elementos de cadeia de caracteres.
* Os dicionários são denotados por colchetes.
* As chaves requerem imutabilidade e exclusividade.
* Os valores podem ser imutáveis ou mutáveis e permitem duplicatas.
* O senhor separa cada par chave-valor com uma vírgula e pode usar o destaque de cores para tornar a chave mais visível.
* O senhor pode atribuir dicionários a uma variável.
* O senhor usa a chave como um argumento para recuperar o valor correspondente.
* O senhor pode fazer adições e exclusões nos dicionários.
* O senhor pode executar uma operação em um dicionário para verificar a chave, o que resulta em uma saída verdadeira ou falsa.
* O senhor pode aplicar métodos para obter uma lista de chaves e valores em um dicionário.

## Conjuntos

* Os **conjuntos** em Python são coleções de elementos exclusivos, úteis para tarefas como a remoção de duplicatas e a realização de operações de conjunto como união e interseção. Os conjuntos não têm ordem.
* Os colchetes `{}` são úteis para definir os elementos de um conjunto.
* Os conjuntos não contêm itens duplicados.
* Uma lista passada pela função `set` gera um conjunto com elementos exclusivos.
* O senhor usa "Set Operations" para executar ações como adicionar, remover e verificar elementos em um conjunto.
* O senhor pode combinar conjuntos usando o operador E comercial `&` para obter os elementos comuns de ambos os conjuntos.
* O senhor pode usar a função `Union` para combinar dois conjuntos, incluindo os elementos comuns e exclusivos de ambos os conjuntos.
* O método de subconjunto é usado para determinar se dois ou mais conjuntos são subconjuntos.
