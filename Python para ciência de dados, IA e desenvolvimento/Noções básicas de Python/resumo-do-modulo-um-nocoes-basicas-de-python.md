# Resumo do Módulo 1: Noções Básicas de Python

## Tipos de Dados

* O Python pode distinguir entre tipos de dados como **inteiros**, **floats**, **strings** e **booleanos**.
* Os **inteiros** são números inteiros que podem ser positivos ou negativos.
* Os **floats** são números que têm pontos decimais; eles podem representar valores inteiros ou fracionários.
* Você pode converter inteiros em floats usando *typecasting* e vice-versa.
* Você pode converter inteiros e floats em strings.
* É possível converter um inteiro ou um float em um booleano: `0` torna-se `False`, diferente de zero torna-se `True`.

## Expressões e Variáveis

* As **expressões** em Python são uma combinação de valores e operações usadas para produzir um único resultado.
* As expressões executam operações matemáticas, como adição, subtração, multiplicação e assim por diante.
* Podemos usar `//` para executar a divisão de números inteiros, que resulta em um valor inteiro descartando a parte fracionária.
* O Python segue a **Ordem das operações (BODMAS)** para realizar operações com várias expressões.
* As **variáveis** armazenam e manipulam dados, permitindo o acesso e a modificação de valores em todo o seu código.
* O **Operador de atribuição** `=` atribui um valor a uma variável.
* A atribuição de outro valor à mesma variável substitui o valor anterior dessa variável.
* Você pode executar operações matemáticas em variáveis usando a mesma variável ou variáveis diferentes.
* A modificação do valor de uma variável afetará outras variáveis somente se elas fizerem referência ao mesmo objeto mutável.

## Operações de String

* As **operações de String** de texto do Python envolvem a manipulação de dados de texto usando tarefas como indexação, concatenação, fatiamento e formatação.
* Uma **string** geralmente é escrita entre aspas duplas ou simples, incluindo letras, espaço em branco, dígitos ou caracteres especiais.
* Uma cadeia de caracteres pode ser atribuída a uma variável e é uma sequência ordenada de caracteres.
* Os caracteres em uma cadeia identificam seus números de **índice**, que podem ser positivos ou negativos.
* As cadeias de caracteres são sequências que suportam operações como **indexação** e **fatiamento**.
* Você pode inserir um valor de `stride` para executar o fatiamento ao operar em uma string.
* Operações como **concatenação** e **replicação** produzem novas cadeias de caracteres, enquanto a determinação do comprimento de uma cadeia de caracteres retorna um número.
* Não é possível modificar uma cadeia de caracteres existente; elas são **imutáveis**.
* Você pode usar **sequências de escape** com uma barra invertida (`\`) para alterar o layout de uma cadeia de caracteres. (Por exemplo, `\n` para uma nova linha, `\t` para uma tabulação e `\\` para uma barra invertida etc.)
* No Python, você executa tarefas como pesquisa, modificação e formatação de dados de texto com seus **métodos de string** predefinidos.
* Você aplica um método a uma string para alterar seu valor, resultando em outra string.
* É possível executar ações como alterar o `case` de caracteres em uma cadeia de caracteres, substituir itens em uma cadeia de caracteres, localizar itens em uma cadeia de caracteres e assim por diante usando métodos de cadeia de caracteres predefinidos.
